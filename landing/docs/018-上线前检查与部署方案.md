## 四、部署方案对比

Landing 为纯静态站点（`output: 'export'`），所有平台均支持秒级部署。

| 平台 | 优点 | 缺点 | 推荐度 |
|------|------|------|--------|
| **Vercel** | 与 Next.js 同厂，自动 HTTPS，全球 CDN，CLI 一键部署 | 国内访问速度一般 | 推荐 |
| **Cloudflare Pages** | 全球 CDN 极快，免费额度高，支持自定义域名 | 需 Cloudflare 账号 | 推荐 |
| **Netlify** | 拖拽部署，自动 HTTPS，表单功能 | 国内速度一般 | 备选 |
| **GitHub Pages** | 免费，与仓库绑定，CI/CD 自动部署 | 国内访问慢，无 CDN | 备选 |
| **阿里云 OSS + CDN** | 国内访问最快，支持备案域名 | 需配置 CDN，有费用 | 国内首选 |
| **腾讯云 COS + CDN** | 同上 | 同上 | 国内首选 |

**决策建议**：
- 若目标用户以海外/开发者为主 → **Vercel**（5 分钟完成部署）
- 若目标用户以国内为主 → **阿里云 OSS + CDN**（需备案，若域名未备案则选 Vercel）
