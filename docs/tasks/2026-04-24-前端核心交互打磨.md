## 项目概述
SparkBin 是一个面向 Vibe 开发者的项目创意管理工具，采用 Brutalist（粗野主义）设计风格。
- 前端：React 18 + Vite + TypeScript + Tailwind CSS + Zustand
- 路由：react-router-dom（BrowserRouter）
- 状态管理：Zustand（projectStore + aiStore）
- 拖拽：@dnd-kit/core + sortable
- 测试：Playwright E2E（已配置，自动启动前后端）

## 本次任务目标
