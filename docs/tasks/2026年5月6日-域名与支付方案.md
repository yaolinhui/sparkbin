## 二、Zeabur 支付方案（中国大陆）

### 现状

Zeabur 使用 **Stripe** 作为支付网关，支持的支付方式：
- Visa / Mastercard / American Express 信用卡
- 部分地区的借记卡
- **不支持**：支付宝、微信支付、银联（除非该银联卡开通了 Visa/Mastercard 双币通道）

### 中国大陆用户的可行支付方案

| 方案 | 难度 | 成功率 | 说明 |
|------|------|--------|------|
| **国内双币信用卡**（Visa/Mastercard） | 低 | 高 | 招商银行、工商银行等发行的双币信用卡，直接绑 Stripe |
| **虚拟信用卡**（OneKey / Depay） | 中 | 中高 | 需要 USDT 充值，适合没有实体外币卡的用户 |
| **PayPal 绑定国内卡** | 中 | 中 | 部分国内银行卡可通过 PayPal 走 Stripe（不一定成功）|
| **找海外朋友代付** | 低 | 极高 | 最省事，把钱转给朋友 |
| **Wise / Revolut 虚拟卡** | 中高 | 高 | 需要护照开户，有海外身份的用户适用 |

### 推荐方案（按优先级）

**第一优先：国内双币信用卡**
- 如果你有招商银行 Visa 全币卡、工商银行 Mastercard 等，直接在 Zeabur Billing 页面绑定
- 计费时按美元结算，银行自动换汇
- 月费通常 $5-20，一年也就几百块人民币

**第二优先：OneKey 虚拟信用卡**
- 官网：https://onekey.so
- 支持用 USDT（加密货币）充值，然后生成 Visa 虚拟卡
- 可以绑定 Stripe 支付 Zeabur
- 注意：需要已有加密货币，且有 KYC（身份认证）

**第三优先：找朋友代付**
- 如果有朋友在海外，直接让对方用信用卡订阅，你人民币转给对方
- 最简单、最稳定

### Zeabur 费用参考

| 套餐 | 价格 | 适合阶段 |
|------|------|---------|
| Free | $0 | 测试/极少量用户 |
| Developer | ~$5/月 | MVP 上线初期 |
| Team | ~$20/月 | 有稳定用户后 |
| Enterprise | 定制 | 大规模 |

> 建议：先用 Free tier 部署上线，确认跑通后再升级付费。Free tier 对于 MVP 演示足够。

## 三、部署步骤（更新版）

### Step 1：Zeabur 后端部署

1. 登录 https://dash.zeabur.com
2. 创建 Project → Deploy New Service → 选择 GitHub `sparkbin` 仓库 → `deploy/production` 分支
3. Zeabur 自动识别 `zbpack.json` 和 `docker/Dockerfile.backend`
4. 创建 PostgreSQL（在同一 Project 内）
5. 配置环境变量（见下方清单）
6. 等待构建完成，拿到自动分配的域名（如 `xxx.zeabur.app`）
7. **（可选付费后）** 绑定自定义域名 `api.wanchun.me`

### Step 2：Vercel 前端部署

1. 登录 https://vercel.com
2. Import Project → 选择 `sparkbin` 仓库
3. Root Directory 选 `frontend`
4. Framework Preset 选 `Vite`
5. Build Command: `npm run build`
6. Output Directory: `dist`
7. 环境变量：`VITE_API_URL=https://api.wanchun.me`（或 Zeabur 临时域名）
8. Deploy
9. **（域名配置）** Project Settings → Domains → 添加 `app.wanchun.me`，按提示配置 DNS CNAME

### Step 3：Landing 页部署

1. Vercel 再导入一次，Root Directory 选 `landing`
2. Framework Preset 选 `Next.js`
3. 部署完成后绑定 `landing.wanchun.me`（可选）
