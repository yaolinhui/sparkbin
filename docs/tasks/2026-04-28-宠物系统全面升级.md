# 任务：宠物系统全面升级 —— 动画增强 + 常驻挂件 + 亲密度

**日期**：2026-04-28
**分支**：`feature/pet-system-upgrade`
**关联决策**：DEC-006（六阶段工作流）

---

## 一、任务目标

将 SparkBin 宠物系统从"聊天皮肤"升级为有情感连接、有成长性、有传播力的数字伙伴系统。

---

## 二、具体实施内容

### Phase 1：像素动画增强
- 补充 `PixelPet.frames.ts` 中各宠物的 `blink` 和 `happy` 多帧动画
- `PixelPet.tsx` 支持新的动画状态（sleepy, celebrate）

### Phase 2：亲密度系统
- 扩展 `AIPetConfig` 类型：增加 `affection`, `totalInteractions`, `firstMetAt`
- 后端 `schemas.py` 和 `auth.py` 支持新字段
- 前端在 AI 聊天、点击互动时自动增加亲密度
- 亲密度解锁新台词等级

### Phase 3：ProjectBoard 常驻挂件增强
- 每日首次问候（根据时间：早安/午安/晚安）
- 项目状态感知气泡（空项目、停滞提醒、完成庆祝）
- 亲密度视觉反馈（宠物挂件旁显示亲密度等级）

### Phase 4：阶段完成庆祝
- `ProjectDetail.tsx` 完成阶段时触发宠物 `celebrate` 动画
- 弹出庆祝气泡和台词

### Phase 5：闭合循环
- `npm run build` 通过
- Playwright 测试通过
- git commit + push
