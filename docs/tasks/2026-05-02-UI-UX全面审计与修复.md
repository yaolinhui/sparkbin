## 项目概述

SparkBin 是一个面向独立开发者的 6 阶段项目工作流工具（IDEA -> VALIDATE -> PROTOTYPE -> SHIP -> GROW -> MONETIZE），采用 Brutalist（粗野主义）设计风格。

- 前端：React 18 + Vite + TypeScript + Tailwind CSS + Zustand
- 路由：react-router-dom v6（BrowserRouter）
- 状态管理：Zustand（projectStore）
- 拖拽：@dnd-kit/core + sortable
- AI 聊天：SSE 流式传输，右侧边栏/全屏双模式
- 设计系统：零圆角（全局 `* { border-radius: 0 !important }`）、JetBrains Mono 等宽字体、CSS 变量 `--brutal-*`

## 本次任务目标

消除 UI/UX 层面的不一致、重复代码、文案混乱和功能冗余，提升整体专业感和可维护性。

### P1-1. 统一按钮样式体系并补齐 :active 状态

**背景**：当前按钮样式存在多种实现方式，且 `.btn-brutal-primary` 缺少 `:active` 状态定义。

**需要做的**：
1. 在 `frontend/src/index.css` 中：
   - 为 `.btn-brutal-primary` 添加 `:active` 状态：
     ```css
     .btn-brutal-primary:active {
       background-color: var(--brutal-accent-hover);
       border-color: var(--brutal-accent-hover);
     }
     ```
   - 为 `[data-theme="light"] .btn-brutal-primary:active` 添加对应样式。

   - 修改 `.btn-brutal:active` 只改背景色和边框色，不改文字色。

   - 使用 className 替代纯内联样式，例如：
     ```tsx

     style={{ backgroundColor: petColor, opacity: !input.trim() || isThinking ? 0.5 : 1 }}
     ```
