# SparkBin 项目 Agent 规则

## ⚠️ 绝对禁止：停止服务

**未经用户明确、具体的书面授权，禁止停止、重启、中断或以任何方式终止任何正在运行的服务、进程、服务器或容器。**

- 包括但不限于：后端 API 服务、前端开发服务器、数据库服务、任务队列、WebSocket 连接等。
- **原因**：停止服务将导致当前环境中所有运行中的语句、会话状态、内存数据全部丢失，且可能造成不可逆的数据损失。
- **执行测试、调试、构建或其他任何任务时，必须优先寻找不中断现有服务的方法。**
- 如果某项任务确实需要停止服务，必须先向用户明确说明风险，并获得用户的明确同意后方可执行。
