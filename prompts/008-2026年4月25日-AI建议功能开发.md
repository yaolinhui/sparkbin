## 项目概述
SparkBin 是一个面向 Vibe 开发者的项目创意管理工具。
- 前端：React 18 + Vite + TypeScript + Tailwind CSS + Zustand
- 后端：FastAPI + SQLAlchemy + SQLite（开发）/ PostgreSQL（生产）
- AI 代理模式：前端不接触 API Key，全部走后端代理

## 本次任务目标
修复项目详情页"想法"阶段的"AI 建议"按钮：点击无反应/报错、不填充面板、丢失用户原始想法。

## 具体工作内容

### 1. 后端：数据库与 API（最高优先级）

#### 1.1 Project 模型新增 original_idea 字段
- 文件：`backend/app/models.py`
- 在 `Project` 表中新增 `original_idea = Column(Text, default="", nullable=False)`

#### 1.2 Schema 更新
- 文件：`backend/app/schemas.py`
- `ProjectCreate` / `ProjectUpdate` / `ProjectInfo` / `ProjectDetail` 都加上 `original_idea: str = ""`

#### 1.3 项目路由更新
- 文件：`backend/app/routers/projects.py`
- 创建项目时接收并保存 `original_idea`
- 更新项目时允许更新 `original_idea`
- 返回项目详情时包含 `original_idea`

#### 1.4 新增 AI 建议端点
- 文件：`backend/app/routers/ai.py`
- 新增 `POST /ai/idea-suggest`
- 请求体：`{ project_id, title, pain_point, original_idea, current_notes: [{title, content}] }`
- 响应体：`{ notes: [{title, content}] }`
- 调用 `ai_service.generate_idea_suggestions(...)`

#### 1.5 AI 代理服务新增方法
- 文件：`backend/app/services/ai_proxy.py`
- 新增 `async def generate_idea_suggestions(self, title, pain_point, original_idea, current_notes, provider, model) -> dict`
- Prompt 要求：忠实于用户原始意图，返回 JSON 格式结构化数据，包含 5 个维度（核心痛点、目标用户、使用场景、解决方案、差异化价值）

#### 1.6 SQLite 兼容性补丁
- 文件：`backend/app/main.py`
